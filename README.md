## Local Installation

1. Clone repository
2. Go to project directory
3. Install dependencies

    ```bash
    npm install
    ```

4. Run app locally

    ```bash
    npm run dev
    # OR
    node index.js
    ```

### Build with:

-   [Node.js](https://nodejs.org/en/)
-   [Express.js](https://expressjs.com/)
-   [MongoDB](https://www.mongodb.com/)
